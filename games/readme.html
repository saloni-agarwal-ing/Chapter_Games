Drop Game

Java & libGDX Project Overview

A simple desktop game built with libGDX, generated using gdx-liftoff.
Demonstrates cross-platform game development in Java.

Project Structure

  • core: Shared game logic and assets
  • lwjgl3: Desktop launcher using LWJGL3

Modular design for easy extension to other platforms.

Build & Run

  • Uses Gradle for dependency management
  • Run with ./gradlew lwjgl3:run
  • Build runnable jar with ./gradlew lwjgl3:jar

Supports IntelliJ IDEA and Eclipse project generation.

Features & Tech

  • Java 17+ required for latest plugins
  • Simple ApplicationAdapter extension
  • Easy to containerize with Docker
  • Potential for multiplayer with KryoNet

How to Enhance

  • Add levels, power-ups, and obstacles
  • Implement online multiplayer and scoreboard
  • Integrate cloud saves and analytics
  • Modernize UI/UX and add sound/music
